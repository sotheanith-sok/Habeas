<!-- Description -->
A complete implementation of large scale search engine including on-disk indexing, multiple queries options, and user interfaces.

## Features
- [Core]
  - Corpus to handle Json
  - Building positional index
  - Indexing and Tokenization
  - Query Language
  - Main Application
  - JsonFileDocument
- [Extra]
  - Unit Testing
  - GUI
  - Wildcard Queries
  - SoundEx
  - Near Operator
- [Features Details]
